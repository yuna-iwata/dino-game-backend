# dino-game-backend

To start:

- Navigate to the directory containing this README
- Create and activate venv:
```
python3 -m venv venv
source ./venv/bin/activate
```
- Install requirements:
```
pip3 install -r requirements.txt
```
- Run API:
```
export FLASK_APP=api.py
export FLASK_DEBUG=1
flask run
```
